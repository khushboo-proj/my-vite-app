# My Vite React App

This is a React-based dashboard application built using Vite.
It contains multiple modules such as patients, doctors, appointments,
inventory, and purchase history with sidebar navigation.

## Features
- Sidebar-based navigation
- Patient management module
- Doctor & appointment sections
- Inventory and medicine purchase tracking
- Clean and modular UI

## Tech Stack
- React
- Vite
- JavaScript (ES6)
- CSS
